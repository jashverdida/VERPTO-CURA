# Asset Placeholders

This folder should contain the following asset files for Expo:

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (1242x2436) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
4. **favicon.png** (48x48) - Web favicon

## Quick Setup

For development, you can use placeholder images. Create simple solid emerald (#10B981) backgrounds with the CURA logo.

## Recommended Tools

- [Figma](https://figma.com) - Design tool
- [Expo Icon Generator](https://github.com/expo/expo/tree/main/packages/@expo/prebuild-config) - Generate all icon sizes

## NOTE for Expo Go

Expo Go will work without these assets, but you'll see placeholder images. For production builds, replace these with your actual branded assets.
